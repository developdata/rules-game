Hello there

This is a group excercise to help you think about designing games and play. You will be creating a set of rules for a game in a different way. Once each person in your group has opened this page on their browser you are ready to play.

Decide an order of play, the person who is going first writes 3 rules for a game in the input fields below, and submits those rules. The other players will only see the last rule they have written.

The next person writes their rules and submits them. Once everyone has written their rules click on the reveal all rules button to see what everyone has written, then design a game around those rules.

When everyone has written 3 rules

Now you have a set of rules for a game you can work out the physical stucture of the game. Think about how you would make this as a physical computing project. How would the players interact, would there be a board, what would the game look like.

If you want to play everyone just refreshes their browser page to start again.
